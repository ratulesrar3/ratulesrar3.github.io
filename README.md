# ratulesrar3.github.io

Personal portfolio site for Ratul Esrar — fractional analytics for healthtech and marketplace startups.

**Live:** https://ratulesrar3.github.io

## Tech

Vanilla HTML, CSS, and JavaScript. No build step, no framework, no dependencies beyond Google Fonts. Clone and open `index.html`.

## Structure

```
├── index.html          # main page
└── assets/
    ├── css/style.css   # all styles
    └── js/viz.js       # genre crossover interactive visualization
```

The genre crossover viz plots real Spotify audio features (tempo × energy) across five genres and surfaces bridge tracks — songs that sit in a neighboring genre's pocket.
